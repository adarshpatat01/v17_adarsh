Extract Invoices, Bills, Credit Notes and Debit Notes Odoo Apps

Split and Extract Invoices/Bills/Credit Notes/Debit Notes Odoo App helps users to configure or allow access to split/extract invoice. Users can easily check or uncheck all invoice/bill/credit/debit lines using buttons and also manually select invoice/bill/credit/debit lines. Using split function users can easily split selected order lines and create a new order and remove selected lines from the existing order and using extract function easily extract selected order lines without removing selected lines from the existing order.

Features

Configure Extract/Split Invoice

Under user configuration tick on allow to extract/split invoice.

Check or Uncheck button on Invoice Lines

Using check or uncheck button, User can easily select or unselect all invoice/bill/credit/debit lines.

Easy to Split Invoice/Bill/Credit Note/Debit Note

User can easily split selected order lines and create a new order and remove selected lines from the existing order.

Easy to Extract Invoice/Bill/Credit Note/Debit Note

User can easily extract selected order lines and create a new order without removing selected lines from the existing order.

Configuration for Allow Extract/Split Invoice

Under user configuration you can tick on "Allow Extract/Split Invoice" checkbox.

Split/Extract Invoice Flow

Click on Check Button to Select All Products

When you click on check button, All products selected of an invoice lines and ready to extract or split.

Click on Uncheck Button to Unselect All Products

When you click on uncheck button, All products unselected of an invoice lines.

Manually Check or Uncheck Products on Invoice Lines

Select Products and Click on 'SPLIT' Button

You can select products from invoice lines and click on 'SPLIT' button so invoice split wizard will be open.

Invoice Split Wizard

When open invoice split wizard, You can select 'New' or 'Existing' split option.

Display Splited Button in Invoices

After splited an invoice, You can see splited button in invoices.

Splited Invoice in List View

After Splited Invoice, New Invoice Generated

After splited invoice, You can see new invoice is generated and also display splited invoice number.

Warning/Validataion

When you click on 'SPLIT' OR 'EXTRACT' button without creating an invoice lines then warning raised.
