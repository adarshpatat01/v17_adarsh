POS Products Tags and Tag Filter Odoo App

Apply Product Tag Filter on POS Odoo Apps

Point of Sales Product Tag with Filter Odoo Apps

POS Products Tags Advance odoo app helps users to add tags on products and filter products in point of sale by added tags. Users can select multiple tags at a time and filter products by multiple tags and can remove all selected tags easily, users can also search orders via filter with tag in a single click. Users can also select existing tags to product or create and add new tags to product from the point of sale screen.

Features

Filter Products Based on Tags

User can filter products based on added tags to products.

Create, Add or Remove Tags for Product from POS Screen

User can create new tag, add existing tag to products or remove existing tag from pos screen.

Remove All Selected Filter with Single Click

User can remove all selected tags with single click at a time.

Assign Multiple Tags to Products

User can assign multiple tags to single products.
Demo Request Video Tutorial

Product Tags Configuration

Under Point of Sale > Configuration > Point of Sale user need to enable "Enable Product tag" to add and filter products by tags.

Create Product Tag

Under Point of Sale > Products > Product By Tag menu user can create tags for products.

Add Product Tags

Under Point of Sale tab of product variants user can add tags for products.

Products Tags on POS Screen

On point of sale screen user can see created tags for products as bellow image.

Filter Products By Tags

On clicking particular tag user can see filtered products by tags.

Filter Products By Multiple Tags

User can also select multiple tags at a time and filter products.

Remove Multiple Tags Selection with Single Click

User can remove multiple tags selection with single click by clicking remove tags icon as shown bellow image.

Add Product Tags from POS Screen

To add tags on product user need to double click product on pos screen.

Select Existing Tag for Product

On double clicking product new wizard will open, Here user can select existing tags for product and click "Add" button.

Create and Add New Tag for Product

User can create and add new tag for product by entering tags name and click "Add and Create" button.

User can see success message on adding or creating new tags for product.

Added Tags form POS Screen

User can see filtered product by added or new created tags.

User can see created tags also added to product in backend.

Remove Tag for Product

User can see selected tags on product.

On double clicking product new wizard will open, Here user can select existing tags for product and click "Remove" button.

User can see success message on removing tags for product.

User can see tags from product also removed in backend.
